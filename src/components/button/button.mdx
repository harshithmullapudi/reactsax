## Default

The buttons are a fundamental part of any project, with vuesax you can add a great button with a single line of code the buttons have all the states as active, focus, hover, to make its implementation simpler and speed up the work

<Playground>
  <section class="d-section">

    </div>

    </div>
    <div class="d-section__item">

    </div>
  </section>
</Playground>

## Flat

You can change the full style towards flat of a button with the flat property that its value is aboolean, so just adding it will change the styles

<Playground>
  <section class="d-section">
    <div class="d-section__item">
      <Button styleButton="flat" active>
        Active
      </Button>
    </div>
    <div class="d-section__item">
      <Button styleButton="flat">Default</Button>
    </div>
    <div class="d-section__item">
      <Button styleButton="flat" disabled>
        Disabled
      </Button>
    </div>
  </section>
</Playground>

## Size

Change the size of the entire button including padding, font-size and border with the size property

- values:
  - xl
  - l
  - default
  - small
  - mini
